# Session Management

Session management helps keep history searchable, context controllable, and issues traceable.

## What You Can Do

- list sessions with search and channel filters
- inspect chat history
- edit session metadata (`label`, `preferredModel`)
- clear session history
- delete sessions

## When It Helps Most

- multiple channels are active and you need to locate one thread quickly
- one session context is polluted and needs a reset
- you want long-running sessions organized by topic

## Beginner Suggestions

1. add simple `label` naming for common sessions
2. clear broken sessions before retrying
3. clean up stale sessions regularly
