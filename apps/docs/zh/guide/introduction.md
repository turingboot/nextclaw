# NextClaw 介绍

**一个先跑起来、再慢慢变强的本地 AI 助手。**

NextClaw 运行在你自己的机器上。你可以在浏览器里完成配置，然后在本地或常用聊天渠道里和它对话。

## 为什么适合新手

- 上手快：`nextclaw start` 一条命令即可启动
- 以 UI 为主：常用配置都能在页面里完成
- 兼容生态：可借鉴 OpenClaw 的大量教程与项目

## 你可以先做到什么

- 配置一个 Provider 和默认模型
- 在 UI 里发第一条消息
- 接入一个常用渠道（如 Discord/Telegram）
- 配一个最小自动化提醒（Cron）

## 快速开始

```bash
npm i -g nextclaw
nextclaw start
```

浏览器打开 `http://127.0.0.1:55667`。
