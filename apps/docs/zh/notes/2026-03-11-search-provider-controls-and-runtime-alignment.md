# 2026-03-11 · 集成博查搜索 + 运行时对齐

发布时间：2026-03-11  
标签：`new feature` `improvement`

## What changed

- 在运行时配置中集成 Bocha（博查）作为可用 Web 搜索提供商。
- 在运行时配置中新增可配置的 Web 搜索提供商支持。
- 在 UI 中新增搜索提供商启用与管理入口。
- 对齐运行时包行为与导出路径，减少开发/运行环境差异。

## Why it matters

- 可直接在产品链路中使用博查搜索，无需自行打补丁接入。
- 你可以按需切换搜索行为，而不必改内部实现。
- 搜索能力对非技术用户更可见、更易配置。
- 运行时链路更稳定，降低环境差异导致的问题。

## How to use

1. 打开 UI 配置页。
2. 进入搜索相关设置，将搜索提供商设为 Bocha（或其他提供商）。
3. 保存后执行一次带搜索能力的任务，确认行为符合预期。
